# Emby Live TV Horizontal Cards

> A custom CSS for Emby that changes the vertical cards to horizontal.

---

## Installation

Add this line to the **Admin Dashboard → General → Custom CSS** part in Emby:

```css
@import url('https://cdn.jsdelivr.net/gh/Mub1532/emby-horizontal-livetv-cards-css@main/index.css');
```

---

## About

This was made since emby only allows you to set it horizontal for the main section when you click into it, not other sections, like home page, or tv guide etc so the tv channel thumbnails look weird by default.
